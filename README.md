دليل تشغيل أداة DeepCheck AI

أولاً: تجهيز بيئة العمل (Backend)

تأكد من تثبيت لغة Python على جهازك.

افتح الـ Terminal في مجلد المشروع واكتب الأمر التالي لتثبيت المكتبات اللازمة:
pip install -r requirements.txt

افتح ملف app.py وقم بوضع مفاتيح الـ API الخاصة بك (Gemini, Sightengine, Hugging Face).

قم بتشغيل السيرفر عن طريق الأمر: python app.py.

ثانياً: تثبيت الإضافة (Extension)

افتح متصفح Google Chrome واذهب إلى الرابط: chrome://extensions/.

قم بتفعيل وضع المطور (Developer Mode) من أعلى اليمين.

اضغط على زر "Load unpacked" واختار مجلد المشروع.https://github.com/leqaa-nsy/DeepCheck-AI-Extension/releases/download/v1.0/MyExtension.zip

ثالثاً: طريقة الاستخدام

اذهب إلى أي موقع (مثل يوتيوب).

قف بالماوس فوق أي صورة أو فيديو، ستظهر لك أيقونة الفحص الحمراء.

اضغط عليها وانتظر التقرير النهائي من الذكاء الاصطناعي.
