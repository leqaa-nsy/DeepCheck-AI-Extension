عبارة عن إضافة لمتصفح جوجل كروم (Google Chrome Extension) صُممت لتكون خط الدفاع الأول للمستخدم أثناء تصفح الإنترنت. المشروع بيستخدم تقنيات الذكاء الاصطناعي لاكتشاف المحتوى المزيف (Deepfakes) ومحاولات الاحتيال (Scams) فور حدوثها، بهدف تعزيز الثقة الرقمية وحماية خصوصية المستخدمين.(يعمل علي كروم فقط لحين التطوير)
لميزات الرئيسية (Key Features):
Deepfake Detection: تحليل الصور والفيديوهات على صفحات الويب للتأكد من أصالتها وكشف التلاعب الناتج عن الذكاء الاصطناعي.

Real-time Scam Alerts: نظام تنبيه فوري بيحذر المستخدم عند الدخول على روابط مشبوهة أو التعامل مع محتوى احتيالي.

Seamless Integration: الإضافة بتشتغل في الخلفية بدون ما تلاحظ أي ثقل في المتصفح، وبتقدم واجهة مستخدم (UI) بسيطة وواضحة.

High Performance: تم بناء الكود ليكون سريعاً جداً في معالجة البيانات وضمان استجابة النظام للتهديدات في أجزاء من الثانية
لجانب التقني (Technical Stack):
Frontend: JavaScript (ES6+), HTML5, CSS3.

Core: Chrome Extension API.

Security Logic: خوارزميات مخصصة للتحقق من سلامة المحتوى والروابط.
ليه المشروع ده مهم؟ (The Impact):
مع انتشار التزييف العميق، أصبح من السهل خداع الناس بصور وفيديوهات غير حقيقية. Omni-Scanner Pro بيحل المشكلة دي عن طريق توفير أداة تقنية قوية وسهلة الاستخدام لأي شخص، حتى لو ملوش في التكنولوجيا، عشان يحمي نفسه من النصب الإلكتروني.

دليل تشغيل أداة DeepCheck AI

أولاً: تجهيز بيئة العمل (Backend)

تأكد من تثبيت لغة Python على جهازك.

افتح الـ Terminal في مجلد المشروع واكتب الأمر التالي لتثبيت المكتبات اللازمة:
pip install -r requirements.txt

افتح ملف app.py وقم بوضع مفاتيح الـ API الخاصة بك (Gemini, Sightengine, Hugging Face).

قم بتشغيل السيرفر عن طريق الأمر: python app.py.

ثانياً: تثبيت الإضافة (Extension)

افتح متصفح Google Chrome واذهب إلى الرابط: chrome://extensions/.

قم بتفعيل وضع المطور (Developer Mode) من أعلى اليمين.

اضغط على زر "Load unpacked" واختار مجلد المشروع.
لينك المجلدhttps://github.com/leqaa-nsy/DeepCheck-AI-Extension/releases/download/v1.0/MyExtension.zip
يجب فك ضغط المجلد اولا
ثالثاً: طريقة الاستخدام

اذهب إلى أي موقع (مثل يوتيوب).

قف بالماوس فوق أي صورة أو فيديو، ستظهر لك أيقونة الفحص الحمراء.

اضغط عليها وانتظر التقرير النهائي من الذكاء الاصطناعي.
